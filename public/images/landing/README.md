# Landing page images

Place your photos in this folder using **exact filenames** below. The site shows placeholders until each file exists.

| File | Suggested content | Recommended size |
|------|-------------------|------------------|
| `hero.png` (or `hero.jpg`) | Rider or van delivering a package | 1200×1000 px |
| `gallery-delivery.png` | Handoff / happy customer | 800×600 px |
| `gallery-tracking.png` | Phone with map / tracking UI | 800×600 px |
| `gallery-rider.png` | Rider on bike or van | 800×600 px |
| `gallery-warehouse.png` | Packages at hub or warehouse | 800×600 px |
| `step-quote.png` | Creating a shipment / laptop | 800×600 px |
| `step-pay.png` | Mobile payment | 800×600 px |
| `step-track.png` | Map or delivery in progress | 800×600 px |
| `roles-team.png` | Team, riders, or operations | 900×1100 px |
| `cta-background.jpg` | Fleet, city, or logistics wide shot | 1600×900 px |

Formats: `.jpg`, `.jpeg`, `.webp`, or `.png` (update filenames in `src/components/landing/landing-images.ts` if you use `.webp`).
